# Roo Code 3.39.3 Release Notes (2026-01-10)

This release renames Roo Code Cloud Provider to Roo Code Router and improves CLI and internal organization.

<img src="/img/v3.39.3/v3.39.3.png" alt="Roo Code v3.39.3 Release" width="600" />

## Roo Code Router rename

Roo Code now shows **Roo Code Router** anywhere the extension previously referred to the **Roo Code Cloud Provider**. The new name is meant to make the intent clearer: Roo Code Router is a router-style provider (similar in idea to other router/aggregator services), but with a **curated set of models weâ€™ve tested with Roo Code**. ([#10560](https://github.com/RooCodeInc/Roo-Code/pull/10560)).
